# Python Backend Learning

This directory contains Python learning materials organized by topic.

## Structure

- **`01-fundamentals/`** - Core Python concepts (functions, scope, lambdas, higher-order functions)
- **`02-practice/`** - Problem-solving exercises (array manipulation, algorithms)

## How to Use

1. Start with `01-fundamentals/functions.py` to understand Python basics
2. Move to `02-practice/questions.py` to solve problems
3. Run files with: `python filename.py`

## Topics Covered

### Fundamentals
- Function definition and calling
- Function scope and global variables
- *args and **kwargs for flexible parameters
- Lambda functions and anonymous functions
- Higher-order functions
- Filter, map, and sorting operations

### Practice Problems
- Finding min/max elements
- List manipulation (sum, average, reverse)
- Counting even/odd numbers
- Removing duplicates
- Array operations

## Prerequisites

- Python 3.8+
- Basic understanding of variables and control flow

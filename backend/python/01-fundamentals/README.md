# Python Fundamentals

Core Python concepts including functions, scope, lambdas, and higher-order functions.

## Files

- **`functions.py`** - Comprehensive guide to Python functions
  - Basic function definition and calling
  - Function scope and global variables
  - *args for variable positional arguments
  - **kwargs for variable keyword arguments
  - Lambda functions and anonymous functions
  - Higher-order functions and callbacks
  - Filter operations with lambda

## How to Run

```bash
python functions.py
```

## Learning Path

1. Understand basic function definition
2. Learn about function scope and global keyword
3. Master *args and **kwargs for flexible parameters
4. Explore lambda functions for concise anonymous functions
5. Understand higher-order functions (functions that take functions as arguments)

## Key Concepts

- **Function Definition**: `def func_name(parameters):`
- **Global Variables**: Use `global` keyword to modify global scope
- **\*args**: Capture multiple positional arguments as a tuple
- **\*\*kwargs**: Capture multiple keyword arguments as a dictionary
- **Lambda**: Anonymous function syntax `lambda args: expression`
- **Higher-order Functions**: Functions that accept or return functions

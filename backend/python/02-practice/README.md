# Python Practice Problems

Hands-on practice problems for array manipulation and fundamental algorithms.

## Files

- **`questions.py`** - Collection of array manipulation problems
  - Finding minimum and maximum elements
  - Finding second largest element
  - Calculating sum and average
  - Counting even and odd numbers
  - Removing duplicates from arrays
  - Reversing arrays

## How to Run

```bash
python questions.py
```

## Problems Included

### 1. Find Largest Number
- Find the maximum element in an array
- Concept: Linear search, comparison

### 2. Find Smallest Number
- Find the minimum element in an array
- Concept: Linear search, comparison

### 3. Find Second Largest
- Find the second largest element
- Concept: Tracking multiple values, comparison

### 4. Sum and Average
- Calculate sum and average of array elements
- Concept: Accumulation, mathematical operations

### 5. Count Even and Odd Numbers
- Count how many even and odd numbers exist
- Concept: Modulo operator, counting

### 6. Remove Duplicates
- Remove duplicate values from array
- Concept: Sets, unique values

### 7. Reverse List
- Reverse array elements
- Concept: Array manipulation, slicing

## Difficulty Level

**Beginner** - All problems are designed for beginners learning fundamental algorithms.

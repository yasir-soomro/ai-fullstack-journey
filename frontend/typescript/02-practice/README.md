# TypeScript Practice

Hands-on practice exercises for TypeScript.

## Files

- **`practice.ts`** - TypeScript exercises and challenges

## How to Use

1. Open `practice.ts`
2. Work through exercises
3. Compile with `tsc`
4. Check compiled output in `../build/`
5. Run with `node ../build/practice.js`

## Topics

Practice exercises cover:
- Type annotations
- Type inference
- Union and intersection types
- Generics
- Type guards
- Advanced types

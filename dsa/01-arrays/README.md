# Array Algorithms

Fundamental array manipulation and algorithm problems.

## Files

- **`practice.py`** - Collection of array algorithm problems

## Topics Covered

- Finding elements (min, max, second largest)
- Array manipulation (reversing, sorting)
- Counting operations
- Duplicate handling
- Searching algorithms

## How to Run

```bash
python practice.py
```

## Difficulty Level

**Beginner** - All problems are designed for learning array basics.

## Problem Categories

1. **Search Problems** - Finding specific elements
2. **Transformation Problems** - Modifying arrays
3. **Counting Problems** - Aggregating array data
4. **Utility Problems** - Common operations

## 🚀 Quick Start

**Python Fundamentals:**
```bash
cd backend/python/01-fundamentals
python functions.py
```

**DSA Practice:**
```bash
cd dsa/01-arrays
python practice.py
```

**TypeScript:**
```bash
cd frontend/typescript
tsc && node build/01-basics.js
```

**FastAPI:**
```bash
cd backend/fastapi/01_fastapi
uvicorn main:app --reload
```

## 📖 Recommended Learning Path

1. **Start Here**: `backend/python/01-fundamentals/` - Python basics
2. **Practice**: `backend/python/02-practice/` - Solve problems
3. **Algorithms**: `dsa/01-arrays/` - Learn data structures
4. **Frontend**: `frontend/typescript/01-fundamentals/` - Type systems
5. **Backend**: `backend/fastapi/01_fastapi/` - Web APIs
6. **Advanced**: Databases, AI Agents (coming soon)
